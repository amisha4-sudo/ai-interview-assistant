# 🚀 AI Interview Assistant

## Description
This project is an AI-based interview assistant that helps users practice interview questions and get feedback.

## Features
- AI-based answer evaluation  
- Simple UI  
- Real-time feedback  

## Tech Stack
- Python  
- Streamlit  
- OpenAI API  

## Project Structure
-app.py
-requirements.txt
-README.md

## Installation
pip install -r requirements.txt

## Run the Project
streamlit run app.py

## How It Works
- User enters question and answer  
- AI processes input  
- Feedback is generated  

## Future Improvements
- Voice input  
- Better AI model
